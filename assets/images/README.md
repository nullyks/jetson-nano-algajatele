# Kaamerapiltide privaatsus

Päris seadmega tehtud kaamerapilte ei lisata vaikimisi avalikku reposse. Pilt võib kogemata näidata inimest, kodu või tööruumi, dokumente, ekraane või muud isiklikku teavet.

Õppematerjalis kasuta ainult pilti, mille avaldamiseks on luba ja millel ei ole tundlikku sisu. Vajadusel hägusta või lõika pilt enne jagamist ning kontrolli ka pildifaili metaandmeid.
